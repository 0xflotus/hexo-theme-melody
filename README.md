# Installation

Find your hexo work folder

```bash
git clone https://github.com/Molunerfinn/hexo-theme-melody themes/melody
```

If you don't have jade & stylus renderer, follow this:

```bash
npm install hexo-jade-renderer hexo-stylus-renderer
```
